OntoVerbal
==========

OntoVerbal is a Protege 4.2 plugin that generates natural language descriptions for classes for an ontology written in OWL (roughly the OWL EL profile). OntoVerbal was written by Fennie Liang, in collaboraitn with Donia Scott, Alan Rector and Robert Stevens, as part of the EPSRC funded Semantic Web Authoring Tool Project (EP/G032459/1).
