# IMDB-movie-review-classifier
Using a pretrained wikitext language model and its embedding to create a sentiment classifier for IMDB movie review.

It achieved the state of the art result. 

The AWD LSTM paper is here. [https://arxiv.org/pdf/1708.02182.pdf]

Using fastai and pytorch to create a AWD-LSTM language model and classifier model for sentiment analysis. 

I translated it into Chinese and uploaded it to kaggle too.
