A dialogue model capable of chatting with a human via Telegram

This model has a transformer structure and is trained on the OpenSubtitles 
dataset

OpenSubtitles dataset can be found here: http://opus.nlpl.eu/OpenSubtitles-v2018.php

Transformer description found in paper Attention Is All You Need
(https://arxiv.org/abs/1706.03762 )

Create Telegram bot which takes in conversation history and produces response using
fully-trained Transformer ( https://github.com/python-telegram-bot/python-telegram-bot )
