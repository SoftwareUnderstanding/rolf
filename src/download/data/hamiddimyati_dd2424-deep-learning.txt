# DD2424-deep-learning
All codes and reports for assignments of deep learning course

Assignment 1:<br/> 
a. Building a single-layer neural network to analyse the CIFAR-10 datasets<br/>
b. Investigate the role of learning rate<br/>
c. Investigate the role of lambda (L2 regularization parameter)<br/>
d. Apply a random shuffling of mini-batch<br/>
e. Apply a learning rate decay<br/>
f. Apply grid-search for hyperparameter tuning<br/>
g. Replace the cross-entropy loss with SVM loss function<br/>
training accuracy: 42.24%<br/>
testing accuracy: 40.89%<br/>

Assignment 2:<br/> 
a. Building a two-layer neural network to analyse the CIFAR-10 datasets<br/>
b. Apply a cyclical learning rate (https://arxiv.org/pdf/1506.01186.pdf)<br/>
c. Apply coarse-to-fine random search for hyperparameter tuning<br/>
d. Add more hidden nodes<br/>
e. Apply dropout (http://jmlr.org/papers/volume15/srivastava14a/srivastava14a.pdf)<br/>
training accuracy: 63.46%<br/>
testing accuracy: 54.10%<br/>

Assignment 3:<br/>
a. Building a k-layer neural network to analyse the CIFAR-10 datasets<br/>
b. Apply He initialization (https://arxiv.org/pdf/1502.01852.pdf)<br/>
c. Apply coarse-to-fine random search for hyperparameter tuning<br/>
d. Add more hidden layers<br/>
e. Apply hidden size tuning<br/>
f. Apply dropout<br/>
training accuracy: 61.34%<br/>
testing accuracy: 55.09%<br/>
