# Caffe YOLO

[![License](https://img.shields.io/badge/license-BSD-blue.svg)](LICENSE)

This Repository combines multiple Deep Learning innovations together within the Caffe Framework.

Caffe is developed by Berkeley AI Research ([BAIR](http://bair.berkeley.edu))/The Berkeley Vision and Learning Center (BVLC) and community contributors. 

- [Project site](http://caffe.berkeleyvision.org)
- [Source code](https://github.com/BVLC/caffe)

Further additions to this repo include
- [Depthwise Separable Convolutions](https://arxiv.org/abs/1704.04861)
- [ShuffleNet Units](https://arxiv.org/abs/1807.11164)
- [YOLO object detection](https://pjreddie.com/darknet/yolo/)

Open source implementations used for this work include

- [caffe-mobilenet](https://github.com/farmingyard/caffe-mobilenet)
- [ShuffleNet](https://github.com/farmingyard/ShuffleNet)
- [Darknet](https://github.com/pjreddie/darknet)
