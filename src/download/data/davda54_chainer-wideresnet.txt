# WideResNet in Chainer

Implementation of WideResNet for CIFAR10 dataset in Chainer. Contains data standardization and augmentations (horizontal flip, translation, cutout). Also label smoothing and weight decay.

You can try it yourself on [**Google Colab**](https://colab.research.google.com/drive/1PhgOQIm4ELQq6ju-mlsUePOJk8yKvvmY)
