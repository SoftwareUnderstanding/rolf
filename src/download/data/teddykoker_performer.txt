# Performers

Simple Numpy implementation of the FAVOR+ attention mechanism introduce in
*Rethinking Attention with Performers* [(Choromanski et al.,
2020)](https://arxiv.org/abs/2009.14794), and code to replicate the experiments
in my article: [Performers: The Kernel Trick, Random Fourier Features, and
Attention](https://teddykoker.com/2020/11/performers/).
