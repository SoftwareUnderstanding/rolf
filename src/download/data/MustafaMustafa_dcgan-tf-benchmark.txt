# DCGAN TF Benchmark

A benchmark TF implementation of DCGAN architecture ([arXiv:1511.06434](https://arxiv.org/abs/1511.06434)).

- - - 
### Usage:  
```bash
git clone git@github.com:MustafaMustafa/dcgan-tf-benchmark.git
cd dcgan-tf-benchmark
sh download.sh
```

You can either run a single test using:  
```bash
python run_dcgan.py
```

Or, scan different parameters:  
```bash
python benchmark_dcgan_scan.py
```
