# stack-overflow-question-classification
Classify stack overlow questions as high quality, low quality - delete, or low quality - edit using a LSTM model and BERT!


DistilBERT, a distilled version of BERT: smaller, faster, cheaper and lighter:
* Able to achieve 93% accuracy on the validation dataset using DistilBERT
* https://arxiv.org/abs/1910.01108
