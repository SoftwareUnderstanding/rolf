# GCN
TensorFlow 2.0 implementation of graph classification using Graph Convolutional Network (SEMI-SUPERVISED CLASSIFICATION WITH GRAPH CONVOLUTIONAL NETWORKS, https://arxiv.org/pdf/1609.02907.pdf), global pooling and DiffPool (HIERARCHICAL GRAPH REPRESENTATION LEARNING WITH DIFFERENTIABLE POOLING, https://arxiv.org/pdf/1806.08804.pdf).
