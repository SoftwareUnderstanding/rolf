# pix2pix-NNFL

Implementation of the pix2pix paper (https://arxiv.org/abs/1611.07004) done as a part of an assignment of the Neural Networks and Fuzzy Logic (NNFL) at BITS Pilani. Implemented using Eager Execution in Tensorflow.

Facades Dataset: http://cmp.felk.cvut.cz/~tylecr1/facade/
Tensorflow version: >= 1.10
