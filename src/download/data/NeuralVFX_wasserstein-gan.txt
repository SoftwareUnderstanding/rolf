# Wasserstein-Gan
This is an implimentation of Wasserstein Gan (https://arxiv.org/pdf/1701.07875.pdf).

I've tested this using the [LSUN bedroom dataset](https://www.kaggle.com/jhoward/lsun_bedroom/downloads/lsun_bedroom.zip/1).
