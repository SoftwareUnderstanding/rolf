# PPOv1
My first implementation of OpenAI's Proximal Policy Optimization(https://arxiv.org/abs/1707.06347). Still learning

# Requirements:
OpenAI's Gym
OpenAI's Baselines - Vectorized Environments
Pytorch
Numpy
TensorboardX for debugging
Matplotlib

# Training
To train run:
```
python trainer.py
```
