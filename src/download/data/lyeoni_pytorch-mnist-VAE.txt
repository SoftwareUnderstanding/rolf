# pytorch-mnist-VAE
Variational AutoEncoder on the MNIST data set using the PyTorch


## Dependencies
- PyTorch
- torchvision
- numpy

## Results
Generated samples from 2-D latent variable with random numbers from a normal distribution with mean 0 and variance 1
