# computer_vision_final_projectRCNN

System requirements

1. Preinstall cuda_9.0.176 for win10
2. Preinstall cudnn-9.0 win10-x64-v7
3. Preinstall Microsoft Visual C++ Build Tools
4. pip install -r requirements.txt
5. python setup.py install

output files:
All output files in the patches folder which contain video detection and image segementations
