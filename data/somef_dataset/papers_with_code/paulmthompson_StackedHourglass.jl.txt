# StackedHourglass.jl

This is a Julia implementation of a stacked hourglass neural network for pose detection. This model was first described in Newell et al 2016: <br>
https://arxiv.org/abs/1603.06937

An excellent PyTorch implemention can be found here: <br>
https://github.com/princeton-vl/pytorch_stacked_hourglass
