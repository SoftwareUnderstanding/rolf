# Deep-Learning-paper-implementations
Implementations of Deep Learning papers using PyTorch and TensorFlow 2.0/Keras

# Papers used
* [LeNet-5 paper](http://yann.lecun.com/exdb/publis/pdf/lecun-01a.pdf)
* [AlexNet paper](http://www.cs.toronto.edu/~hinton/absps/imagenet.pdf)
* [VGG net paper](https://arxiv.org/abs/1409.1556)
* [ResNet paper](https://arxiv.org/abs/1512.03385)
* [Neural style transfer paper](https://arxiv.org/abs/1508.06576)
* [YOLO Object Detection paper](https://arxiv.org/abs/1506.02640)
