# Word2VecCustomModel
Code for basic functionality  Word2Vec model  and how you can train your custom word2vec model using genism in python 

Pre-requisites:

Python 3.6   https://www.python.org/downloads/

gensim https://pypi.org/project/gensim/  Version < 4.0

sci-kit learn https://scikit-learn.org/stable/install.html

pandas (pip install pandas)
