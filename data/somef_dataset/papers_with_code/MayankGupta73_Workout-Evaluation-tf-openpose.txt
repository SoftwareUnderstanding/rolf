# Workout Evaluation Using Deep Learning

Body posture is very important during excercise. Incorrect posture can lead to inefficent workouts and aching/complications in the body.<br>
This is an application which will correct body posture during various workouts.<br>
It will compare the position of various limbs of the user with corresponding position of a trainer in realtime using feed from a webcam.
The user can find out whether their posture was correct or not and see their errors.<br>
Various workouts can be loaded via videos from trainers.
