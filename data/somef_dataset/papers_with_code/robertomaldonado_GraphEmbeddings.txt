# GraphEmbeddings
The project includes has the original implementations for the following network embedding techniques.
* [1] DeepWalk
* [2] Node2Vec
* [3] Struc2Vec

Please refer to the original papers:

[1] https://arxiv.org/abs/1403.6652

[2] https://arxiv.org/abs/1607.00653
