## Foot counter 
This can count people realtime in the videos build with opnencv and caffe2 and used deep learning model Mobilenet/SSD object detection.
Some references:
- https://www.pyimagesearch.com/2018/08/13/opencv-people-counter/
- https://caffe2.ai/
- https://arxiv.org/abs/1512.02325
