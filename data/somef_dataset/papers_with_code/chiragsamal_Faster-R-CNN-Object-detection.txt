# Faster-R-CNN-Object-detection


Faster R-CNN Object Detection using PyTorch

In this project, a Faster Region-based Convolutional Network method (Faster R-CNN) for object detection is implemented using PyTorch.

Official Paper: [Faster R-CNN: Towards Real-Time Object Detection with Region Proposal Networks](http://papers.nips.cc/paper/5638-faster-r-cnn-towards-real-time-object-detection-with-region-proposal-networks.pdf)
