# DCGAN for CIFAR-10

This notebook has very simple and easy to follow implementation for Deep Convolutional Generative Adversarial Network (DCGAN) for CIFAR10 image generation in tensorflow 1.x.

This easy to use Jupyter notebook can be used on Google Colab
