# Attention-Augmented-Convolutions
Refer to this paper: https://arxiv.org/pdf/1904.09925.pdf

This is an (ongoing) implementation of the paper "Attention Augmented Convolutional Networks" by Bello et al, written using Pytorch.
The code has been adapted from the Tensorflow code provided in the original paper.
