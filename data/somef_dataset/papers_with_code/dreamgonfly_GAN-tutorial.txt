# GAN-tutorial
Tutorial on GAN and DCGAN in PyTorch

## GAN.ipynb
Linear 레이어를 이용하여 MNIST에 GAN을 학습시킵니다.

Paper : [https://arxiv.org/abs/1406.2661](https://arxiv.org/abs/1406.2661)

### GAN_generated_images
GAN을 학습시키며 만들어진 생성 결과를 담는 폴더입니다.
이미지들을 연결하여 만든 .gif도 같이 담깁니다.

## DCGAN.ipynb
Convolution과 Transposed convolution을 이용하여 MNIST에 DCGAN을 학습시킵니다.

Paper : [https://arxiv.org/abs/1511.06434](https://arxiv.org/abs/1511.06434)

### DCGAN_generated_images
DCGAN을 학습시키며 만들어진 생성 결과를 담는 폴더입니다.
이미지들을 연결하여 만든 .gif도 같이 담깁니다.
