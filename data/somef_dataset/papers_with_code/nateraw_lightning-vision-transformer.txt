# lightning-vision-transformer

A quick [PyTorch Lightning](https://github.com/PytorchLightning/pytorch-lightning) implementation of the [Vision Transformer](https://arxiv.org/pdf/2010.11929v1.pdf) (ViT).

This code is based on [lucidrain's ViT Repo](https://github.com/lucidrains/vit-pytorch) - with some :fire: edits by [@teddykoker](https://github.com/teddykoker)
