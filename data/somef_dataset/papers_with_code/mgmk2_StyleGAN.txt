# StyleGAN
TF2.0 implimentation of StyleGAN (https://arxiv.org/abs/1812.04948).
This implimentation works on CPU, GPU and also colab TPU.

# Requirements
* Python3
* Tensorflow >= 2.0.0
* Numpy
* Pillow
* Matplotlib
* Pandas
