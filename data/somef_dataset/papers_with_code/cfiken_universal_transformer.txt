# Universal Transformer

This is unofficial implementation of Universal Transformer [arXiv:1807.03819](https://arxiv.org/abs/1807.03819) by TensorFlow Eager Execution and Keras API.
The official implementation is available in [tensor2tensor](https://github.com/tensorflow/tensor2tensor).
