# GANS Implementation

My Tensorflow implementation of GANs as presented in https://arxiv.org/abs/1406.2661 

## Required Libraries

You can use package manager like [pip](https://pip.pypa.io/en/stable/) to install.

```bash
tensorflow
cv2
```

## Usage

- Download data for [CiFar](http://www.cs.utoronto.ca/~kriz/cifar.html), [MNIST](http://yann.lecun.com/exdb/mnist/) and [Toronto Face Dataset (TFD)](https://vjywpjj.tk/sports/toronto-face-dataset.php) into corresponding sub-folders in the 'Data' Folder.
- Run  the 'train_net.py' script
