# Car_Detection
 Car and empty parking area detection on parking lot drone images with YOLOv4

# Dataset - CARPK Dataset

The Car Parking Lot Dataset (CARPK) contains nearly 90,000 cars from 4 different parking lots collected by means of drone (PHANTOM 3 PROFESSIONAL). The images are collected with the drone-view at approximate 40 meters height.

https://paperswithcode.com/dataset/carpk

# Object Detection - YOLOv4

You only look once (YOLO) is a state-of-the-art, real-time object detection system.

https://arxiv.org/pdf/2004.10934.pdf

https://pjreddie.com/darknet/yolo/
