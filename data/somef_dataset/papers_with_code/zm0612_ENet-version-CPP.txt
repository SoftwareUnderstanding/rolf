This is the implementation of the C++ version of enet!

## Publication
This work has been published in arXiv: [`ENet: A Deep Neural Network Architecture for Real-Time Semantic Segmentation`](https://arxiv.org/abs/1606.02147).

## Instructions
### build
  
  `cd /segmentation`
  
  `mkdir build`
  
  `cd build`
  
  `cmake ..`
  
  `make`

### run
  `cd /example`
  
  `./segment`
