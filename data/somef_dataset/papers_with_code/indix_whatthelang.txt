## Basic Usage

Predicting Language using ``whatthelang``

```python
>>> from whatthelang import WhatTheLang
>>> wtl = WhatTheLang()
>>> wtl.predict_lang("Mother")
'en'
>>> wtl.predict_lang("‡Æ§‡Ææ‡ÆØ‡Øç")

>>> wtl.predict_lang("‡¥Ö‡¥Æ‡µç‡¥Æ")

```

Batch Prediction is also supported

```python

```
