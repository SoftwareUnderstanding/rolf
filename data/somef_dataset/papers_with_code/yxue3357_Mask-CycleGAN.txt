
# Mask-CycleGAN
An simple implementation of Mask-CycleGan using pytorch.

# Notes
    1. 在做马和斑马的实验时，斑马->马 很容易，得到的mask也很规整，但马->斑马却很十分困难。
    2. 在桔子和苹果数据集上效果挺好。
