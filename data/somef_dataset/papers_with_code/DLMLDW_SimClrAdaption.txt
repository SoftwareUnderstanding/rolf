Implementation of SimClr in tensorflow 2.0
Code adapted from: https://github.com/sayakpaul/SimCLR-in-TensorFlow-2
Code used from: https://github.com/marrlab/instantdl
Following this paper: https://arxiv.org/abs/2002.05709

To run it put images in a folder and specify the path in the main.py
Set the batchsize, image dimensions and epochs there. 
Execute the main.py
