# quora-ques-pair-similarity-BERT

Enthused by the recent advancements in NLP, I was curious to understand and run BERT pre-trained model on the Quora Question Pair Similarity detection dataset to experience the bi-directionality of BERT. This colab notebook is a step-by-step guide to the implementation of the same using pytorch-transformers by hugging face.
