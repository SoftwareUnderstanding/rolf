# DCGAN-Face-Generation

This is a Udacity project which aims to train a DCGAN with Celeba dataset and produce faces that does not exist.

DCGAN paper can be found in here: https://arxiv.org/abs/1511.06434

## Installation

Make sure these packages are installed:
- PyTorch 0.4
- Torchvision 
- NumPy

Packages can be installed via PyPi package repository or Anaconda.

## Usage
Notice that you will need to download the dataset mentioned before to proceed for training part. You will also need to train network to use it.

Please follow the Jupyter Notebook file for more information.

Important notice: This GitHub repository includes files for education purposes only. These files should not use for a commercial usage.
