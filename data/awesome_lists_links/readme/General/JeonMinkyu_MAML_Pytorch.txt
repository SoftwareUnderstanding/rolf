#  MAML-Pytorch
PyTorch implementation of the supervised learning experiments from the paper:
Model-Agnostic Meta-Learning (MAML): https://arxiv.org/abs/1703.03400



refer to this github : https://github.com/dragen1860/MAML-Pytorch, https://github.com/lim0606/MAML-Pytorch

# Platform

- Python : 3.7.3
- Pytorch : 1.2.0

# Ominiglot

run ```python main.py``` , the program will download omniglot dataset automatically
