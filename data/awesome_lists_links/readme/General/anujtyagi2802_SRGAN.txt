# Keras-SRGAN
Implementation of SRGAN for Single Image Super Resolution test

# Usage Instructions
python test_new.py -i <input_directory>

# Link to paper
https://arxiv.org/abs/1609.04802

# Training Statistics
I trained the model on Google Colab. The training data included 200 images. I trained the model for 15000 epochs. The GPU used was Tesla K80. It can be improved if the number of images are increased.
