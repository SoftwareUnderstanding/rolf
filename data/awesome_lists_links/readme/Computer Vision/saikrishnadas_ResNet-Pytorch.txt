# ResNet-Pytorch
Pytorch implementation of ResNet50,ResNet101,ResNet152.

This is based on original paper : https://arxiv.org/pdf/1512.03385.pdf

Get started :

python version > 3

pip install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch
