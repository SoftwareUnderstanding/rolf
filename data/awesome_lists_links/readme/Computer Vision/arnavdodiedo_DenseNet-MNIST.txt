# DenseNet-MNIST

An implementation of DenseNet trained on the MNIST handwritten digits dataset.

#### To download and locally store the MNIST dataset, use the following commands:
    from tensorflow.examples.tutorials.mnist import input_data
    input_data.read_data_sets('my/directory')

#### References:
1. Official DenseNet Keras implementation - https://github.com/cmasch/densenet
2. DenseNet Paper - https://arxiv.org/pdf/1608.06993.pdf
