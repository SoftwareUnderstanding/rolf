Present a (Frictionless) dataset for viewing and exploration.

## Install

Git clone then:

```
yarn install
```

## Usage

In this directory:

```bash
export PORTAL_DATASET_PATH=/path/to/my/dataset
yarn dev
```

And you will get a nice dataset page at `http://localhost:3000`
