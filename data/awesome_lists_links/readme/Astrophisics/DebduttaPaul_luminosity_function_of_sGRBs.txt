# luminosity_function_of_sGRBs
This is a repository containing databases and scripts for short GRB luminosity function work, Paul D. 2018, MNRAS, 477, 4275 (https://ui.adsabs.harvard.edu/#abs/2018MNRAS.477.4275P). Please cite this paper if you are using any of these codes or references.

Some codes are similar to those in Paul, D. 2018, MNRAS, 473, 3385 (https://ui.adsabs.harvard.edu/#abs/2018MNRAS.473.3385P), and the corresponding repository is: https://github.com/DebduttaPaul/luminosity_function_of_lGRBs.
