# locus
This repository contains the code and scripts used for the Locus Algorithm software published in Creaner et al, 2019. [1]

This readme is a preliminary version as part of a migration of documentation from .doc format to markdown.  Design documentation is found in [this directory](/Design_Documents/)

[1] O. Creaner, E. Hickey, K. Nolan, N. Smith The locus algorithm, (In Preparation).
