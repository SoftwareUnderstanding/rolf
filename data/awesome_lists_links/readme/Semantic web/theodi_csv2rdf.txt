# CSV 2 RDF

A ruby gem to convert CSV to RDF, following the CSV on the Web specification.


## Installation
    $ gem install specific_install
    $ sudo gem specific_install -l https://github.com/theodi/csv2rdf
    
## Usage

    $ csv2rdf myfile.csv
