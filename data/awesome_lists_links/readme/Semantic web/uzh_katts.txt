KATTS
=====

KATTS is a collection of storm operators (bolts) and a configuration facility that enables
a user to write queries in an XML syntax to query a stream of RDF triples.  

Information
===========
- The scripts used for the SSWS/ISWC 2013 publications can be found in src/eval/.
- Information about version numbers can be found in VERSIONS.md
